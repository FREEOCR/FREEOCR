# FREEOCR
永久免费的文字识别（OCR）软件，使用方法如下。

> 重要提醒：不建议充值账号，免费额度基本足够日常使用。本软件会在本地存储您的密钥方便每次打开使用时自动填充。只要无账户余额，即使本地存储泄露也不会有风险！

1. 下载免费软件[Snipaste](https://zh.snipaste.com)。
2. 下载并解压[免费OCR
.zip](https://raw.githubusercontent.com/FREEOCR/FREEOCR/master/zip/免费OCR.zip)，打开`免费OCR软件.exe`。
3. 按照文末的申请方法，填写免费申请使用的密钥。
![a1.png](https://i.loli.net/2020/03/01/5oAlcyJU4xRQ1P6.png)
4. 使用Snipaste截图快捷键`F1`进行截图并复制。
![a2.png](https://i.loli.net/2020/03/01/yhbXeZEKo1HgIm8.png)
5. 点击识别即可获取结果，可使用合并段落功能进行合并。
![a3.png](https://i.loli.net/2020/03/01/kzVpPJH5CosyBQ1.png)


> 密钥存储在您的文档目录下，识别的截图会在桌面保存一份。
